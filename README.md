## Construido con

- React.js y [NESS.css](https://nostalgic-css.github.io/NES.css/). 

## Levantar el proyecto 

- Clonar el repositorio

  `git clone https://github.com/Yeimy7/guess-pokemon-challenge.git`

- Instalar dependencias

  `npm install`

- Levantar el proyecto

  `npm start`
